# Rock Paper Scissors

A simple Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Features

* Play Rock, Paper, or Scissors against the computer.
* Random computer move generation.
* Displays whether you win, lose, or tie.

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Clone this repository.
2. Open `index.html` in your web browser.
3. Play the game by clicking one of the buttons.

## Future Improvements

* Add score tracking.
* Improve the user interface.
* Display results on the page instead of using alerts.

This project was created as part of my JavaScript learning journey to practice functions, conditional statements, event handling, and random number generation.
